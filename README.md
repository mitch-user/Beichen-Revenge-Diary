# 顧北辰重生復仇日記 之霸總歸來

## 專題簡介
本專題為一款短篇視覺小說結合三個小遊戲的互動式遊戲。
玩家扮演重生後的顧北辰，透過劇情推進與關卡挑戰，完成復仇並迎向結局。

## 使用套件
- Python 3.x
- pygame-ce
- pyyaml

## 遊戲玩法
1. 點擊滑鼠推進劇情對話
2. 關卡一：貪食蛇（與敵對蛇搶果實）
3. 關卡二：踩地雷（含增益 Buff）
4. 關卡三：接龍（完成後進入結局）

## 執行方式
```bash
pip install -r requirements.txt
python main.py
